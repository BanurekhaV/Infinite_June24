ASP.net Web Forms : web development platform that provides a programming model, a comprehensive software infrastructure and various services required to build robust web applications for pc, as well as mobile devices

-Introduced in the year 2002 with dot net framework 1.1
-Web Applications have to be hosted only on web servers
-Browser is the client
-Can be accessed anywhere with just internet and browser
-Cross platform
-Asp.net applications built using framework gets hosted automatically onto the IIS(Internet Information Services)

Web Form/Page

-- has 2 files (1. designer file with extension .aspx.designer.cs, and 2. Code behind file with an extension .aspx.cs)

--designer files are autogenerated and we are not supposed to tamper them

--Every web form has 2 modes of operation (design mode and other is Source mode ) .aspx
